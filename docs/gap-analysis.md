# 기획서 대비 구현 현황 및 추가 기능 제안

## 1. 기획서 MVP 기능 구현 현황

| 기획서 MVP 항목 | 구현 상태 | 비고 |
|----------------|-----------|------|
| 회원가입 및 본인 인증 | ✅ 완료 | JWT 인증 + 카카오/네이버/PASS/공동인증서 본인인증 |
| 데이터 항목별 동의/철회 관리 | ✅ 완료 | 카테고리별 동의/철회 + 이력 관리 |
| 건강검진 PDF/이미지 업로드 | ✅ 완료 | 드래그앤드롭 + 파일 타입 검증 |
| OCR 기반 검사 수치 추출 | ✅ 완료 | 시뮬레이션 (실제 OCR 엔진 연동 준비) |
| 혈압, 혈당, 체중, 증상, 복약 수동 입력 | ⚠️ 부분 | 수치 입력 API 있음, 전용 UI 미구현 |
| Apple HealthKit/Google Health Connect 연동 | ✅ 완료 | 연동 상태 + 데이터 시뮬레이션 |
| 검진 결과 요약 카드 | ✅ 완료 | 대시보드 카테고리별 상태 |
| 대사건강 위험 신호 표시 | ✅ 완료 | 정상/주의/경고/위험 4단계 |
| 90일 건강 행동 계획 | ✅ 완료 | 주간 미션 + 진행률 |
| 병원 상담 질문 자동 생성 | ✅ 완료 | LLM 기반 질문 생성 |
| 병원 제출용 PDF 리포트 | ⚠️ 부분 | 리포트 생성 완료, 실제 PDF 렌더링 미구현 |
| 공개 API 기반 병원/약국/의약품 정보 조회 | ❌ 미구현 | |

## 3. 추가 구현 제안 (기획서 외 확장)

### 3.1 높은 우선순위 (사용자 가치 + 기술적 실현 가능)

#### A. 공개 보건 API 연동 (기획서 MVP 누락분)
- **병원/약국 검색**: 공공데이터포털 건강보험심사평가원 API
- **의약품 정보 조회**: 식약처 의약품 안전나라 API
- **내 주변 병원 추천**: 검진 결과 이상 소견 카테고리 기반 진료과 매칭
- **의약품 상호작용 체크**: 현재 복용 약물 간 상호작용 경고

#### B. 건강 타임라인 (진료/검진/투약 통합 뷰)
- 연도별 검진 수치 변화 추이 차트
- 진단 → 처방 → 검사 결과 연결 뷰
- 주요 이벤트 마커 (진단일, 약물 시작일, 생활습관 변경일)

#### C. 수동 입력 전용 UI 강화
- 혈압/혈당/체중 일일 기록 화면
- 복약 체크리스트 (아침/점심/저녁)
- 증상 일지 (두통, 피로, 소화불량 등 선택형)
- 식사/운동 간편 기록

#### D. 실제 PDF 리포트 생성
- WeasyPrint 또는 Puppeteer 기반 PDF 렌더링
- 병원 제출용 표준 양식
- QR 코드로 원본 데이터 검증 링크

### 3.2 중간 우선순위 (차별화 + 리텐션)

#### E. 푸시 알림 / 리마인더 시스템
- 주간 미션 리마인더
- 복약 시간 알림
- 검진 주기 도래 알림
- 위험 수치 변화 감지 시 즉시 알림

#### F. 가족/보호자 모드
- 부모님 계정 연결 (위임 동의)
- 보호자 대시보드 (요약 뷰)
- 이상 징후 시 보호자 알림
- 병원 동행 시 공유 리포트

#### G. 만성질환 관리 프로그램
- 당뇨 전단계 관리 (혈당 트래킹 + 식이 가이드)
- 고혈압 관리 (혈압 기록 + 나트륨 모니터링)
- 이상지질혈증 관리 (콜레스테롤 추적 + 운동 계획)
- 질환별 맞춤 90일 프로그램 자동 생성

#### H. 데이터 내보내기 / 이동권
- FHIR Bundle 형식으로 전체 데이터 내보내기
- CSV/Excel 내보내기
- 다른 플랫폼으로 데이터 이전 지원
- 계정 삭제 시 데이터 완전 삭제 증명

### 3.3 낮은 우선순위 (장기 / 사업 확장)

#### I. 의료진 검수 콘솔 (Admin)
- AI 코치 답변 검수 워크플로우
- 의료 자문위원 리뷰 대시보드
- 부적절 답변 신고/차단 관리
- 의학 가이드라인 RAG 소스 관리

#### J. B2B2C 기업 복지 패키지
- 기업 관리자 대시보드 (익명 통계)
- 임직원 검진 후 사후관리 자동 배정
- 부서별/연령별 건강 트렌드 리포트
- 기업 맞춤 건강 챌린지

#### K. 유전체 데이터 연동
- DTC 유전자검사 결과 업로드 (23andMe, 마크로젠 등)
- 유전적 위험 인자 + 검진 수치 교차 분석
- 약물 유전체학 (Pharmacogenomics) 기반 약물 반응 예측 참고

#### L. 연구 참여 / 데이터 기부
- 가명처리 동의 관리
- 연구 프로젝트별 참여 신청
- 데이터 기부 보상 (포인트/기부금)
- IRB 승인 연구만 표시

## 4. 기술적 개선 제안

| 영역 | 현재 | 개선안 |
|------|------|--------|
| 데이터베이스 | In-Memory (Docker에 PostgreSQL 준비) | 실제 PostgreSQL 연동 코드 작성 |
| OCR | 시뮬레이션 | Tesseract.js 또는 Google Vision API 실제 연동 |
| 검색/RAG | 없음 | pgvector 또는 ChromaDB로 의학 가이드라인 RAG 구축 |
| 실시간 알림 | 없음 | WebSocket 또는 SSE 기반 실시간 알림 |
| 모바일 | 웹 전용 | React Native 또는 PWA 전환 |
| 테스트 | 없음 | Vitest + Playwright E2E 테스트 |
| CI/CD | 없음 | GitHub Actions 파이프라인 |
| 모니터링 | 없음 | OpenTelemetry + Prometheus + Grafana |

## 5. 종합 우선순위 제안

즉시 구현 가능하면서 사용자 가치가 높은 순서:

1. **공개 보건 API 연동** (병원/약국/의약품 검색) - 기획서 MVP 누락분
2. **수동 입력 전용 UI** (혈압/혈당/체중 일일 기록) - 리텐션 핵심
3. **건강 타임라인** (연도별 수치 변화 추이) - 데이터 가치 시각화
4. **실제 PDF 리포트 생성** - 병원 제출 실용성
5. **푸시 알림 / 리마인더** - 리텐션 개선
6. **가족/보호자 모드** - 초기 고객 확장 (부모님 건강 관리)
7. **만성질환별 관리 프로그램** - 차별화
